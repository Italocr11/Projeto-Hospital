# Projeto-Hospital
## Sistema de Gestão Hospitalar - Hospital Prime
### Descrição<br>
Este projeto tem como objetivo o desenvolvimento e modelagem de um banco de dados para um sistema de gestão de um hospital particular.<br>

### Funcionalidades
O sistema permite o controle de:
- pacientes
- médicos
- funcionários
- Responsáveis
- consultas
- internações
- exames
- prescrições
- faturamento
- convênios
- setores
- leitos
- agendamento de consulta
- resultado de exame
- medicamentos
